Instances of Lit Boxes with Materials and Textures

The code is heavily based on the WebGL Programming Guide by Kouichi Matsuda and Rodger Lea. You can find more information about the book and the source code on their webpage. The code posted here is with the generous permission of the authors.

This demo uses a VAO, UBO, array attributes, and instanced rendering in WebGL 2.

Please use a browser that supports "canvas".
Use up/down arrows to change speed.
